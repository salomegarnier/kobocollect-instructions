# KoboCollect Instructions Generator

A simple static web app that helps KoboToolbox project admins generate tailored KoboCollect instructions for data collectors.

The tool asks a few questions about the KoboCollect project settings enabled for a project, then generates the relevant field instructions based on those settings.

## What it does

The generator creates customized instructions for common KoboCollect workflows, including:

- Downloading forms
- Collecting data
- Editing drafts
- Sending finalized forms to the server
- Retrieving form updates

The generated instructions change depending on the selected project configuration, including:

- Blank form update mode
- Automatic download
- Auto send

## How to use

Open the web page, answer the questions under **KoboCollect project settings**, then click **Generate instructions**.

You can then:

- Copy the generated instructions
- Print the instructions
- Share them with data collectors before or during training
